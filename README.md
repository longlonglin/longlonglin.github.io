# Longlong Lin Academic Homepage

This repository hosts the GitHub Pages source for Longlong Lin's academic homepage:

https://longlonglin.github.io

Core site pages:

- Home: `_pages/about.md`
- Publications: `_pages/publications.md`
- Team: `_pages/team.md`
- Datasets: `_pages/datasets.md`
- ReadingPaper: `_pages/readingpaper.md`

The site is built with Jekyll/GitHub Pages and keeps the Academic Pages / Minimal Mistakes theme files needed for layout, styling, and rendering.
